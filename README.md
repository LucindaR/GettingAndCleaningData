#README for Getting and Cleaning Data

 Course Project
source data for the project: https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip

##As required, the run_Analysis.R code will:
Merge the training and the test data together
Extract the mean & st dev for each measurement
Use descriptive activity names to name the activities in the dataset
Label dataset with descriptive activity names and 
Create a tidy dataset with the average of each variable for each activity and each subject
